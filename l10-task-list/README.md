# About
Branch created to code along Udemy course: https://www.udemy.com/course/laravel-beginner-fundamentals

# Setup
* Rename `.env.example` to `.env`
* Setup database
* `php artisan key:generate`
* `Install composer`
* `php artisan migrate`
* `php artisan db:seed`
